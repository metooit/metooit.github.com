    学习 Git

    寒假里的日子，有点让人觉得不知到要做什么，我很怕自己浪费时间，所以觉得要学会自己找事情做，然后我觉得有必要认真的学习一下git。Google了一下，找到了这个git官方学习指南，我就照着这个先看一看git吧。：-）

    关于Git

    首先，看一下我的git版本吧：

    git version 1.7.10.4 这应该是最新的版本了吧。

    然后，如何取得帮助呢？

    Git的特点：

    • 几乎可以离线工作，并且很快（相对与SVN来说）
    • 安装体积小
    • 基于snapshot（快照），而不是changesets 如果要更加深入的了解git，请看Git for Computer Scitentists
    • 分支的代价更小，合并方便

    Git的初始化设置

    第一次使用Git时，首先要学会设置好一些东西，比如名字和邮件地址。设置的命令是git config,如果加上--global,这个命令会将设置信息存储在~/.gitconfig。
